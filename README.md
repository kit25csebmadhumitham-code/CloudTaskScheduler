# ☁️ Cloud Task Scheduling System

## 📌 Description

This project implements a cloud-based task scheduling system that manages and executes tasks efficiently based on their priority and execution time. It simulates how cloud systems allocate resources to tasks using optimized scheduling techniques.

---

## 🚀 Features

* Add tasks with ID, priority, and execution time
* Execute highest-priority task first
* Display all tasks in sorted order
* Search tasks using Binary Search
* Efficient scheduling using Priority Queue

---

## 🛠️ Technologies Used

* Java
* Data Structures (Priority Queue, Binary Search)
* Object-Oriented Programming (OOP)


3. Run the program:

```
java Main
```
